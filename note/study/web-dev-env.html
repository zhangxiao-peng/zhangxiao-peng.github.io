前端开发环境配置

Mr.ZhangYp2024年9月25日大约 2 分钟学习环境配置

前端开发环境配置

node.js(JavaScript运行时环境)
官网下载安装包
双击安装包安装nodejs
cmd窗口运行node -v 显示版本号安装成功

node安装后自带npm 但版本不是最新
包管理工具
npm -v 查看npm版本
npm init 初始化
安装 npm install 包  或者npm i 包 
npm i 包 -g全局安装
npm remove 包 卸载
cnpm
	npm install -g cnpm --registry=https://registry.npm.taobao.org
npm淘宝镜像
	npm config set registry https://registry.npm.taobao.org
yarn安装
	npm install yarn -g 或者下载安装包安装
git安装
官网下载安装即可切记安装路径不要使用中文
首次使用配置用户名邮箱
# 配置自己的姓名
git config --global user.name "Your Name" 
# 配置自己的邮箱
git config --global user.email "email@example.com" 
常用基础命令：
	git init 仓库初始化
    git add ----  文件加入暂存区
    git commit -m '说明'  提交到本地仓库
    版本回滚
    	git log
      	git reset --hard 哈希字符串
    .gitignore文件配置忽略文件
    git branch name 创建分支
    git checkout name 切换分支
    git merge name 合并分支
    git checkout -b name 创建并切换
    git push -u origin master 本地仓库内容推送到远程仓库
    git clone https://github.com/xiaohigh/test2.git 克隆
	git pull  更新代码
  协作开发：
  	第一次提交                      第 n 次
    	得到git远程地址密码          更新代码
        将代码克隆到本地             创建并切换
        切换分支                    开发功能
        开发代码                    提交
        本地提交                    合并分支
        合并分支                    更新代码
        更新本地代码                推送代码
        提交代码
vscode编辑器
官网下载安装包即可
安装完成先安装中文插件（chinese language）
之后安装开发必备插件
小程序开发
微信小程序官网注册小程序账号，下载微信开发者工具
使用VScode开发小程序必备插件：
	wxml  微信小程序wxml格式化以及高亮组件   wechat-snippet 微信小程序代码辅助,代码片段自动完成
	vscode wxml  提供wxml语法支持及代码片段    小程序助手
